int * DynamicStack::multiPop ( int  num)

Pops multiple elements of the stack and returns a pointer to the first element.

Warning! This is a dangerous method. It will return a pointer to the first element of an array of elements. This pointer points to memory allocated by the stack. It is not safe to use the elements in the returned array and add elements to the stack at the same time, since the newly added elements will overwrite the old values the pointer is pointing to.
